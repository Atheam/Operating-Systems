Aby skompilowac programu nalezy uruchomic komende
 
	make compile

Na poczatku nalezy uruchomic server ktory bedzie czekal na podlaczajacych 
sie klientow.

Aby klient polaczyl sie z drugim klientem nalezy wpisac polecenie CONNECT
po czym program zapyta uzytkownika o ID drugiego klienta np
CONNECT
Enter id: 5

nie nalezy wpisywac naraz polecenia i ID drugiego klienta tj. CONNECT 5
